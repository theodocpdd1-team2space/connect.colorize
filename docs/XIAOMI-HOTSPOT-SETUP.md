# Xiaomi Hotspot Setup

1. Turn on Xiaomi hotspot.
2. Set hotspot name, for example `EASYCOM-HOTSPOT`.
3. Set hotspot password.
4. Ask all crew to connect to the same hotspot.
5. Open `https://easycom.vjmrtim.my.id`.
6. Join room via QR/link.
7. Allow microphone.
8. Use wired headset.
9. Enable Keep Screen Awake.
10. Keep screen active during critical moments.

EasyCom Web requires internet for website, room, QR, login, and Socket.IO signaling.

Xiaomi hotspot provides both internet and a local Wi-Fi network. For best result, all crew must connect to the same hotspot/router so WebRTC audio can use the best local peer-to-peer path when possible.

Bluetooth/TWS headset may add noticeable delay. Use wired headset for best result and minim delay.

Native app note: for true lock-screen/background communication, EasyCom will need Android/iOS native app support. Web mode is optimized for active-screen usage.
